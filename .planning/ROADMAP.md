# Project Roadmap: NagrikAI UI SaaS Upgrade

## Milestone 1: SaaS Shell & Visual Identity
The goal is to establish the new "Vishwa" brand and the basic dashboard structure for visitor usage.

### Phase 1: Design Tokens & Base Theme
- Implement the Indigo/Saffron theme.
- Set up global typography and spacing tokens.
- Create the base layout shell (SaaS Header + Footer).

### Phase 2: Visitor Dashboard & Document History
- Implement a dashboard view.
- Set up LocalStorage for document history.
- Display "Empty State" with clear instructions.

## Milestone 2: Core Processing UX
Transforming the single-page MVP into a guided multi-step processing flow.

### Phase 3: Simplified Upload & Scan Flow
- "Step 1: Choose Document" flow with camera/file integration.
- "Step 2: Processing" view with engaging progress feedback.

### Phase 4: Result View & Audio Integration
- Redesign the summary page for better legibility on mobile.
- Integrated "Listen" button with playback progress.

## Milestone 3: Localization & Accessibility

### Phase 5: Multilingual UI Localization
- Implement i18n framework.
- Localize all UI strings.

### Phase 6: Final Visual Audit & Accessibility Polish
- WCAG 2.1 AA check.
- Staggered entry animations for a "premium" feel.
